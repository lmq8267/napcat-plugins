# 随机回应表情插件

NapCat QQ 机器人插件，自动回应指定用户的消息，随机发送选中的 QQ 表情。

## 功能特性

- **启用开关**: 可随时启用/禁用插件
- **多用户支持**: 可配置多个需要回应的 QQ 号
- **QFace 表情选择**: 内置完整 QQ 表情数据，使用模态弹窗选择表情
- **随机回应数量**: 可配置随机回应几个表情，0 表示回应全部选中的表情
- **指定 QQ 群**: 支持多选群组，不选表示回应所有群聊
- **日志输出**: 回应时在日志中输出提示，包括回应用户的消息内容、表情 ID 和表情名称

## 安装

### 方式一：从 GitHub Release 下载

1. 从 [GitHub Releases](https://github.com/lmq8267/napcat-plugins/releases) 下载最新版本的 `napcat-plugin-emoji-reply.zip`
2. 解压到 NapCat 的 `plugins` 目录

### 方式二：本地构建

```bash
# 安装依赖
pnpm install

# 构建插件
pnpm run build
```

将生成的 `dist/` 目录复制到 NapCat 的 `plugins` 目录。

## 使用方法

1. 在 NapCat WebUI 中进入 **插件管理** > **随机回应表情** > **配置**
2. 启用插件开关
3. 添加需要回应的用户 QQ 号（可添加多个）
4. 点击「打开表情选择器」按钮，在弹窗中选择要随机回应的表情
5. 设置随机回应数量（0 表示回应全部选中的表情）
6. （可选）选择指定 QQ 群（可多选），不选表示回应所有群聊
7. 点击 **保存配置**

## 配置说明

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 启用插件 | 开启后自动回应指定用户的消息 | 关闭 |
| 回应用户 QQ 号 | 要回应的用户 QQ 号列表，可添加多个 | 空 |
| 随机回应数量 | 每次随机回应几个表情，0=全部 | 1 |
| 指定 QQ 群 | 指定只回应某些群，可不选或选择多个群 | 全部 |
| 选择表情 | 从内置表情库中选择要随机发送的表情 | 空 |
